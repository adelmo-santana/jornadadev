# Pseudocódigo
## a. Calcular a área de um retângulo:
```
**INICIO**  

Leia Base  
Leia Altura  
Area ← Base * Altura  

Escreva Area  

**FIM**
``` 
## b. Verificar se um número é par ou ímpar:
``` 
**INICIO**  

Leia Numero  

SE Numero % 2 = 0   
Escreva "Par"  

SENÃO  
Escreva "Ímpar"  

FIM SE
**FIM**
```
