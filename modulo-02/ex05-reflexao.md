# Reflexão - O computador não erra?

```
Máquinas apenas seguem instruções que lhes foram dadas, não pensam por sí próprias. Quando uma máquina erra, ela apenas segiu oque nós escrevemos para ela fazer. O erro não foi dela, mas sim nosso, então precisamos pensar cada passo antes de dar a ordem para a máquina, para que ela não leve a culpa de apenas fazer seu trabalho e dizer que a mesma está errada.
```
